# Medaram Smart Crowd Management System

This project is an AI-powered crowd management and safety system designed for the Medaram Jatara festival. The system uses camera feeds, YOLO-based object detection, and data analytics to monitor crowd density, analyze crowd movement, and improve public safety during large gatherings.

## Features

* Camera-based crowd monitoring
* Real-time person detection using YOLO
* Crowd density analysis
* Data visualization and reporting
* Emergency alert system
* Manual crowd reporting when cameras are unavailable or non-functional
* Decision support for authorities

## Technologies Used

* Python
* YOLO (Object Detection)
* OpenCV
* Pandas
* NumPy
* Matplotlib
* Microsoft Excel

## Backup Monitoring System

If cameras are unavailable due to technical issues, power failures, or network problems, authorized personnel can manually report crowd conditions through the system. This ensures continuous monitoring and safety management even when automated surveillance is interrupted.

## Project Goal

To enhance crowd safety and improve event management through AI-powered computer vision, data analytics, and manual monitoring support during large public gatherings such as the Medaram Jatara.
